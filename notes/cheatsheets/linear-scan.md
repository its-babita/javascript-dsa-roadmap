# Linear Scan Pattern

## Idea

Visit every element exactly once.

---

## When to Use

- Find Maximum
- Find Minimum
- Find Sum
- Count Elements
- Search Element

---

## Template

```js
let result = arr[0]

for(let i = 1; i < arr.length; i++) {
  // update result
}
```

---

## Complexity

Time: O(n)

Space: O(1)

---

## Example Problems

- Find Maximum Element
- Find Minimum Element
- Count Even Numbers
- Find Sum
