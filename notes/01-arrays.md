# Arrays -  (DSA JavaScript)

## What is an Array?
A collection of elements stored in a single variable.

Example:
let arr = [10, 20, 30, 40];

---

## Core Idea
We use loops to process arrays element by element.

---

## Important Patterns in Arrays

### 1. Linear Scan
Used for:
- Max/Min
- Sum
- Counting

Template:
- Start from first element
- Loop through all elements
- Update result variable

---

## Problems Practiced

### 1. Find Maximum Element
- Start with first element as max
- Compare with all elements

Time: O(n)
Space: O(1)

---

### 2. Find Minimum Element
Same logic as max but reversed comparison.

---

### 3. Find Sum
- Initialize sum = 0
- Add each element

Time: O(n)

---

### 4. Count Even Numbers
- Check condition num % 2 === 0
- Increment counter

---

### 5. Reverse Array
- Loop from end to start
- Push into new array

Time: O(n)
Space: O(n)
